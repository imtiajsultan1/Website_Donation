Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a tree of objects, allowing developers to access, modify, and interact with HTML and XML content dynamically using JavaScript.

How do you select an element from the DOM?

To select an element from the DOM, you can use methods like document.getElementById('id'), document.querySelector('.class or #id'), or document.getElementsByClassName('class') in JavaScript. These methods allow you to target and manipulate specific elements on a web page.

What is event delegation in the context of DOM, and why is it useful?

Event delegation is a technique where a single event listener on a parent element manages events for multiple child elements. It’s useful for better performance and handling dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element’s attributes and styles in the DOM using JavaScript. Use methods like setAttribute, removeAttribute, and style to modify properties. For example:

  • Attributes: element.setAttribute("class", "new-class");
  • Styles: element.style.color = "red";
